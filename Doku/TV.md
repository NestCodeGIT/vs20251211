| Feature          | مثال                              | توضیح                    |
| ---------------- | --------------------------------- | ------------------------ |
| Namespace        | `namespace AufgabeTV;`            | گروه‌بندی کلاس‌ها        |
| Class            | `internal class TV`               | قالب برای شیء تلویزیون   |
| Access Modifier  | `private`, `public`, `internal`   | محدودیت دسترسی           |
| Field            | `_volume`, `_switchedOn`          | داده‌های داخلی کلاس      |
| Array            | `_programs`                       | نگهداری چند مقدار هم‌نوع |
| Type             | `bool`, `int`, `string[]`         | انواع داده در C#         |
| Method           | `RaiseVolume()`, `TurnOn()`       | عملکردهای کلاس           |
| Parameter        | `SetVolumeStep(int step)`         | ورودی متد                |
| Return type      | `bool`, `void`                    | چیزی که متد برمی‌گرداند  |
| If statement     | `if (!IsOn()) return;`            | تصمیم‌گیری در جریان اجرا |
| Operators        | `+`, `-`, `%`, `!`                | عملیات ریاضی و منطقی     |
| Math methods     | `Math.Min`, `Math.Max`            | محدود کردن مقادیر        |
| Console          | `Console.WriteLine()`             | چاپ روی صفحه             |
| Encapsulation    | `private fields + public methods` | مخفی کردن داده‌های داخلی |
| Local variable   | `string status = ...`             | متغیر موقت داخل متد      |
| Ternary operator | `switchedOn ? "an" : "aus"`       | شرط کوتاه                |
